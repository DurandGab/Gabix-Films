# Gabix-Films




Gabix Films est une application de type bibliothèque de films réalisé en php et js avec les frameworks Laravel et React.
Cette application est en cours de développement et permettra aux utilisateurs de :


- Consulter les films/séries/émissions TV proposés sur l'application 

- D'effectuer des recherches pour trouver les films/séries/émissions TV qu'ils souhaitent regarder

- De créer un compte pour acceder à des fonctionnalités comme ajouter un films/séries/émissions TV ou donner une critique a un(e) films/séries/émissions TV

- De donner une critique à un(e) films/séries/émissions TV de leurs choix

- D'ajouter un nouveau films/séries/émissions TV sur l'application en définissant le titre, les acteurs, la date de parution, l'affiche, le synopsis, etc...

- De consulter les notifications sur son profil (votre films/séries/émissions TV à reçu une critique)


L'administrateur pourra : 

- Valider ou non l'ajout d'un(e) films/séries/émissions TV d'un utilisateur

- Supprimer les films/séries/émissions TV de toute l'application


L'application a été faite dans le cadre d'un devoir, il manque certaines options qui pourront être améliorées dans le futur...
